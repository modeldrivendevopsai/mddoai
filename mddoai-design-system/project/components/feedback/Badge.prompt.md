**Badge** — a small monospace label for version tags, counts, and categories. Quieter than StatusPill and not tied to pipeline state.

```jsx
<Badge variant="brand">v1.0.1</Badge>
<Badge variant="neutral">EPL-2.0</Badge>
<Badge variant="solid">new</Badge>
```

Variants: `neutral`, `brand` (faint violet), `solid` (filled violet), `outline`, `success`. `size="sm"` for inline use.
