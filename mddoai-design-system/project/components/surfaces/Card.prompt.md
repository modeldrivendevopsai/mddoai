**Card** — the standard surface container: white, 1px hairline, soft cool shadow, `--radius-lg`. Use for feature tiles, list items, panels.

```jsx
<Card>…</Card>
<Card interactive onClick={open}>Clickable tile</Card>
```

`hover` adds a lift on hover; `interactive` adds pointer + lift for clickable cards. Never apply a coloured left-border accent — off-brand.
